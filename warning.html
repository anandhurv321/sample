HR Employee Attrition Prediction Model - Input Instructions

Thank you for using our HR Employee Attrition Prediction Model! To make accurate predictions, please follow the instructions below:

  1. Understanding the Columns:
    • You will need to provide values for 14 different columns to make a prediction.
    • Each column represents a specific attribute or characteristic of an employee
  2. Inserting Values:
    • Ensure that you input only numerical values in each column. Do not use any other characters
    • For categorical columns that have been converted to numeric types, such as 'Gender,' use the following mapping:
        0: Female
        1: Male
    • Carefully review each column description before inserting values to ensure accuracy.
  3. Column Descriptions:
    • Column 1: Age - Enter the age of the employee in years.
    • etc....
  4. Submitting Values:
    • Once you have entered values for all 14 columns, click the "Submit" or "Predict" button to receive the attrition prediction.
    • Double-check your inputs to ensure accuracy before submitting.
  5. Interpreting Results:
    • After submitting, you will receive the attrition prediction result based on the input values provided.
    • A positive prediction indicates a likelihood of attrition, while a negative prediction suggests the employee is less likely to leave the company.
  6. Feedback and Support:
    • If you encounter any issues or have questions regarding the input process or predictions, please contact our support team for assistance.

Thank you for using our HR Employee Attrition Prediction Model. We hope this tool provides valuable insights for your workforce management efforts!
